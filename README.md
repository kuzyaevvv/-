# Задание на лабораторную работу
Написать программу на языке C/C++ для перемножения двух квадратных матриц.
Исходные данные: файл(ы) содержащие значения исходных матриц.
Выходные данные: файл со значениями результирующей матрицы, время выполнения, объем задачи.
Обязательна автоматизированная верификация результатов вычислений с помощью сторонних библиотек или стороннего ПО (например на Matlab/Python).
# Требования
**Компилятор**: C++11 или выше.\
**ОС**: Windows, Linux, macOS.\
**Библиотеки**: стандартные библиотеки C++.\
# Состав проекта
*лаба один.cpp - основной код на C++\
*matrix_A.txt - исходная первая матрица для умножения\
*matrix_B.txt - исходная вторая матрица для умножения\
*result_matrix.txt - результат умножения матриц\
*python verify_result.py - код для проверки результата на Python
# Формат входных файлов 
**matrix_A.txt:**\
3\
1 2 3\
4 5 6\
7 8 9\
\
**matrix_B.txt:**\
3\
9 8 7\
6 5 4\
3 2 1
# Обработка ошибок
Программа **лаба один.cpp** выдаёт понятные сообщения при:\
*отсутствии файла;\
*ошибке чтения данных;\
*несовпадении размеров матриц;\
*некорректном формате файла.\
# Формат выходного файла
**result_matrix.txt**\
3\
30 24 18\
84 69 54\
138 114 90
# Скрипт проверки результатов python verify_result.py
Скрипт на **python verify_result.py** проверяет корректность умножения матриц, выполненного программой на C++.
**Что делает:**\
*считывает исходные матрицы из matrix_A.txt и matrix_B.txt;\
*вычисляет их произведение с помощью NumPy;\
*сравнивает результат с выводом программы на C++ (из консоли или файла);\
*выдаёт отчёт совпадают ли результаты или есть расхождение
# Вывод
Проект реализует умножение квадратных матриц: программа на C++ считывает данные из файлов, выполняет вычисление и выводит результат, а скрипт на Python (python verify_result.py) автоматически проверяет корректность результата с помощью NumPy. Решение кроссплатформенное, подходит для учебных и тестовых задач.
Для запуска достаточно собрать C++‑программу, выполнить её и запустить скрипт верификации — он сравнит результат с эталонным и выдаст вердикт, совпадают ли результаты или есть расхождения
